# **Maze of Shadows**



**Developer:**  Nilanga Prabhath Jayawickrama



### 1\. System Requirements



**Operating System**: Windows 10 / Windows 11

**Architecture:** 64-bit

**RAM:** Minimum 4GB

**Graphics:** Integrated graphics supported



**No additional software installation required**



### 2\. How to Run the Game (Executable Version)



Extract the zip file.

Open the folder named:

 	**MazeOfShadows\_Build**

Double-click:

 	**MazeOfShadows.exe**



The game will launch in windowed mode.



**⚠️ Important:**

Do NOT move or delete the **“MazeOfShadows\_Data”** folder.

The .exe file must remain in the same folder as the Data folder.



### 3\. Game Controls



**W / A / S / D** – Move player

**Dash Key (space)** – Dash forward in current direction

**ESC** – Pause Menu



### 4\. Objective



You play as an pilot who has crashed in enemy territory.

Your objective is to navigate through the enemy-built maze, avoid hazards, and reach the exit.



**Hazards include:**



Spikes (booby traps)

Wandering soldiers

Patrol soldiers

Shooter soldiers (watchers)



If you touch any hazard, you will respawn at the starting point.

Reach the exit to win the level.



### 5\. Levels



The game contains:



**Level 1 –** Introductory maze with moderate hazard placement



**Level 2 –** More complex maze with increased hazard difficulty



### 6\. How to Compile the Game (Unity Project Version)



If compiling from the Unity project:



Open the project in Unity (Unity version used: **Unity 6.3 LTS (6000.3.10f1)**).



**Go to:**

File → Build Settings



**Select:**

Platform: PC, Mac \& Linux Standalone



**Target:** Windows



**Architecture:** x86\_64



Ensure the scenes are added in this order:



0 – Main Menu

1 – Level 1

2 – Level 2



Click **“Build”** and choose a folder location.



### 7\. Audio Features



The game includes:



Background music

Pause menu music

Collision sound effects

Death sound

Start sound

Winning sound

Respawn sound



Audio persists correctly between scene transitions.
