# **Maze of Shadows**

**Developer:**  Nilanga Prabhath Jayawickrama<br>


### 1\. System Requirements<br>


**Operating System**: Windows 10 / Windows 11<br>
**Architecture:** 64-bit<br>
**RAM:** Minimum 4GB<br>
**Graphics:** Integrated graphics supported<br>

**No additional software installation required**<br>


### 2\. How to Run the Game (Executable Version)

- Extract the zip file.<br>
- Open the folder named:<br>
 	 **MazeOfShadows\_Build**<br>
- Double-click:<br>
 	 **MazeOfShadows.exe**<br>

The game will launch in windowed mode.

**⚠️ Important:**

Do NOT move or delete the **“MazeOfShadows\_Data”** folder.<br>
The .exe file must remain in the same folder as the Data folder.<br>


### 3\. Game Controls


- **W / A / S / D** – Move player<br>
- **Dash Key (space)** – Dash forward in current direction<br>
- **ESC** – Pause Menu<br>

### 4\. Objective

You play as an pilot who has crashed in enemy territory.<br>
Your objective is to navigate through the enemy-built maze, avoid hazards, and reach the exit.

**Hazards include:**

- Spikes (booby traps)<br>
- Wandering soldiers<br>
- Patrol soldiers<br>
- Shooter soldiers (watchers)<br>

If you touch any hazard, you will respawn at the starting point.<br>
Reach the exit to win the level.<br>


### 5\. Levels

The game contains:<br>

- **Level 1 –** Introductory maze with moderate hazard placement<br>
- **Level 2 –** More complex maze with increased hazard difficulty<br>


### 6\. How to Compile the Game (Unity Project Version)

If compiling from the Unity project:<br>
Open the project in Unity (Unity version used: **Unity 6.3 LTS (6000.3.10f1)**).<br>

**Go to:**
File → Build Settings<br>

**Select:**
Platform: PC, Mac \& Linux Standalone<br>

**Target:** Windows<br>

**Architecture:** x86\_64<br>

Ensure the scenes are added in this order:<br>
0 – Main Menu<br>
1 – Level 1<br>
2 – Level 2<br>

Click **“Build”** and choose a folder location.<br>


### 7\. Audio Features

The game includes:<br>

- Background music<br>
- Pause menu music<br>
- Collision sound effects<br>
- Death sound<br>
- Start sound<br>
- Winning sound<br>
- Respawn sound<br>

Audio persists correctly between scene transitions.
